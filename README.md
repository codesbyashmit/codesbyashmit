## `> projects --production`

<br>

**[ 01 ]&nbsp;&nbsp;CAMPUSCONNECT** &nbsp;—&nbsp; *The Operating System for Student Life*

> `Next.js 15` &nbsp;&nbsp;`Supabase` &nbsp;&nbsp;`TypeScript` &nbsp;&nbsp;`PostgreSQL` &nbsp;&nbsp;`Gemini RAG`
>
> **2nd Place &nbsp;·&nbsp; Annual Hostel Hackathon &nbsp;·&nbsp; 150+ participants**

A full-stack, real-time campus platform that replaced fragmented WhatsApp groups and physical notice boards. Peer matching, SOS alerts, classified marketplace, RBAC admin panel, and a RAG-powered chatbot — one cohesive system.

**[&nbsp;Live Demo&nbsp;](https://campus-connect-three-fawn.vercel.app/)**

<details>
<summary><b>View architecture breakdown</b></summary>

<br>

| Module | Description |
|---|---|
| Vibe Tribe | Intent-based peer discovery and matching |
| Smart Feed | Real-time SOS alerts and Lost & Found |
| Tribe Chats | 1-on-1 and group real-time messaging |
| Classified Marketplace | Campus-scoped buy/sell board |
| Realtime Notice Board | Instant college-wide announcements |
| Admin Panel | RBAC-powered content moderation |
| Campus Genie | RAG chatbot via Supabase Vectors + Gemini |
| Events Hub | College and community event management |

</details>
